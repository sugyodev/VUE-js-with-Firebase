# VuFire

Vue 3 app that signup & login a new firebase user.


![app](https://raw.githubusercontent.com/gshohat/vufire/master/assets/VuFire.gif)

## Project setup
create '.env.local' file in root for firebase config:
```
VITE_API_KEY=
VITE_AUTH_DOMAIN=
VITE_PROJECT_ID=
VITE_STORAGE_BUCKET=
VITE_MESSAGING_SENDER_ID=
VITE_APP_ID=

```

Run
```
npm install
npm run dev
```
